# Geometric Prior-Guided High-Resolution Network (GP-HRNet)

This is the official PyTorch implementation of the paper: **"Geometric Prior-Guided High-Resolution Network for Industrial Micro-Defect Detection"**.

## 📌 Introduction
In this repository, we provide the code for the proposed Geometric Prior-Guided High-Resolution Network (GP-HRNet), a novel visual computing framework designed to detect microscopic defects on Printed Circuit Boards (PCBs) and other complex industrial surfaces.
